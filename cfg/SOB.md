# SOB (Start of Business) — 預設訂票參數

## 預設值

| 參數 | CLI Flag | 值 |
|---|---|---|
| 起程站 | `-f` | `1` (Nangang 南港) |
| 到達站 | `-t` | `5` (Hsinchu 新竹) |
| 出發時間 | `-T` | `3` (06:00) |
| 成人票數 | `-a` | `1` |
| 身分證字號 | `-i` | `Q122821678` |
| 高鐵會員 | `-m` | `y` |
| 手機號碼 | `-p` | `0954747258` |

## 執行指令

```bash
.venv/bin/python -m thsr_ticket.main \
  -f 1 -t 5 -T 5 \
  -i "Q122821678" \
  -m y \
  -p 0954747258
```

> 日期 `-d` 未設預設值，每次執行仍需輸入。
